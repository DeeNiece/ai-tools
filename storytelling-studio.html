How to Use AI StoryTelling Studio

This tool generates multi-episode story series for short-form video platforms. It uses a two-phase approach: first building a Story Bible, then generating each episode script sequentially with continuity. All output saves to your Google Sheet automatically.

Step-by-Step Workflow

1
Set up Google Sheets — Apps Script URL loads automatically from Railway. Expand the Apps Script Setup section to copy and deploy the code if not done yet.
2
Choose your story settings — Genre, Tone, Video Length, number of Episodes, and optionally a Story Seed and Ending Direction for narrative control.
3
Select target platforms — TikTok, YouTube Shorts, Instagram Reels, or Facebook. Social posts will be tailored per platform.
4
Click Generate — AI builds the Story Bible first, then each episode sequentially. Progress bar shows each step.
5
Copy each episode script into HeyGen as a separate talking photo video. Each episode is self-contained but references the previous cliffhanger.
6
Push to Google Sheets — click Push to Google Sheets in the Spreadsheet Preview section. Each episode saves as one row.

Google Sheet Column Reference

Each episode saves as one row. The sheet tab name is set automatically to the genre (e.g. urban legend, horror). If the tab exists it appends.

Column What it contains & how to use it
TimestampWhen this episode was generated.
GenreStory genre selected. Also used as the sheet tab name.
ToneNarrative tone (suspenseful, dark, heartwarming, etc.).
DurationTarget video length. Use this to select the right HeyGen duration.
SeedThe original story seed you entered. Useful for recreating a series later.
EndingThe ending direction you specified. Reference this to maintain series consistency.
HookThe opening hook line for this episode. The first 3–5 seconds that stop viewers from scrolling. Use as your video title card text or opening overlay.
Full_ScriptComplete word-for-word avatar narration script. Paste directly into HeyGen. Includes [[expression]] tags if enabled.
Thumbnail_TextShocking 4–7 word text for the video thumbnail. Add in CapCut or Canva on top of a Gemini-generated background.
PlatformsWhich platforms were selected when generating. Reference for where to post this episode.
Facebook_PostReady-to-post Facebook caption. Copy and post directly.
Instagram_PostInstagram caption with relevant hashtags.
TikTok_PostShort punchy TikTok caption with trending hashtags.
YouTube_TitleSEO-optimized episode title. Copy directly to YouTube upload field.
YouTube_DescriptionFull YouTube description with episode context and series teaser. Paste directly to YouTube.
YouTube_TagsComma-separated tags. Paste into YouTube tags field during upload.

HeyGen Tips for Series Content

Use the same avatar and voice across all episodes for brand consistency. Each episode ends with a cliffhanger — use the Hook from the next episode as a teaser at the end of the current video. Upload episodes as a playlist on YouTube for maximum watch time. On TikTok, post episodes daily at the same time to build anticipation.

Railway Setup

Add APPS_SCRIPT_STORYTELLING in Railway Variables with your deployed Apps Script URL. AI provider keys (DEEPSEEK_API_KEY, GEMINI_API_KEY, PERPLEXITY_API_KEY) are already in Railway — select provider from the AS bar at the top.
