How to Use Avatar News Forge

This tool generates AI avatar news scripts for 4 categories — World Celebrity Gossip, PH Celebrity, AI News, and PH Blind Items. Each category fetches a top-10 list with 2+ sentences per item and no pause tags, ready for HeyGen avatar narration.

Step-by-Step Workflow

1
Set date range — Use the quick buttons (Today, Yesterday, This Week, This Month) or set custom dates. This tells the AI which period to cover.
2
Click Fetch on any news card. The AI generates 10 items, each with 2+ full sentences and no pause tags — ready for avatar reading.
3
Copy the script using the Copy button on each card. This copies the full formatted script including intro, numbered items with ..... separators, and outro.
4
Paste into HeyGen — Use a talking photo avatar. The ..... separators create natural pauses between items without formal pause tags.
5
Send to Sheet — Saves the full script plus AI-generated YouTube teaser, video description, and tags to the correct tab in your Google Sheet.
